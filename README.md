# api-express

An API with Node.js - Express - JSON Web Token

### PUBLIC ENDPOINTS :
- /users --> (GET)
- /groups --> (GET)
- /groups/users --> (GET)
- /register --> (POST)
- /login --> (POST)

### PUBLIC ENDPOINTS :
- /users/:id --> (GET, PUT, DELETE)
- /users/:id/groupe --> (PUT)
- /groups --> (POST)
- /groups/:id --> (PUT, DELETE)
